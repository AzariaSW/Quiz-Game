# Quiz-Game

A console-based Quiz Game System built using Object-Oriented Programming (OOP) principles in Java.
The project demonstrates inheritance, abstraction, interfaces, polymorphism, and real-world design with admin & student accounts, leaderboard, and dynamic question management.

# Features
# User System

Student Signup

Student Login

Admin Login (predefined admin account)

Secure username + password validation

# Student Features

Take quizzes

Receive score at the end

Automatically added to leaderboard

# Admin Features

Add new quiz questions

Add MCQ or True/False questions

View all questions

View leaderboard

Clear leaderboard

Logout safely

#Quiz Features

Multiple-choice questions

True/False questions

Score calculation

Dynamic expandable question set (admin can add more)
